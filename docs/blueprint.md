# **App Name**: CaterEase

## Core Features:

- Service Provider Directory: Display a searchable list of catering service providers, including their logos, phone numbers, and service areas.
- Provider Profile: Detailed profile pages for each provider, including a photo gallery, menus, and a description of their services.
- Rating and Reviews: Enable users to rate and review catering providers.
- AI-Powered Recommendations: Suggest providers based on user preferences (cuisine, budget, event type) using AI. The AI tool must incorporate reviews in deciding on the output.

## Style Guidelines:

- Primary color: Golden Yellow (#FFC107) to convey warmth and culinary excellence.
- Background color: Off-white (#FAFAFA) for a clean and modern look.
- Accent color: Earthy Orange (#D35400) to highlight calls to action.
- Body font: 'PT Sans', sans-serif, suitable for the main text of the app.
- Headline font: 'Playfair', serif, pairing with PT Sans for body.
- Use food and service-related icons.
- Subtle animations on hover and data loading.
